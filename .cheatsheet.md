# Cheat Sheet

## vim

### fold
- zo : 開く
- zc : 閉じる
- zr : 全体を一段開く
- zR : 全体を全て開く
- zm : 全体を一段閉じる
- zM : 全体を全て閉じる

### pane
- vs : 縦に分割
- sp : 横に分割
- <C-e> : 幅調整モードへ移行

### macro
- q(a-z) : 記録開始
- q : 記録終了
- [数字]@(a-z) : 任意の回数分マクロを実行
